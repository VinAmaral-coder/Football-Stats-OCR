# Image-Inspector

Image-Inspector is a Python project focused on extracting football match statistics from FIFA game screenshots using Optical Character Recognition (OCR).

The application identifies player and club statistics, processes the extracted data and exports the results to structured formats such as CSV for further analysis.

## Technologies

- Python
- OpenCV
- Tesseract OCR
- pytesseract
- CSV
- Dataclasses

## Goal

The goal of this project is to automate the extraction of FIFA match statistics through Computer Vision and OCR, transforming screenshots into structured datasets that can be analyzed, stored or exported.
